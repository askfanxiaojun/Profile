## 如何写新文章

在 `content/blog/` 目录下新建一个 `.mdx` 文件，文件名就是 URL 路径。

每篇文章顶部需要填写以下信息（frontmatter）：

```yaml
---
title: "文章标题"
description: "文章简介（显示在列表卡片上）"
date: "2024-03-18"
categories: ["分类1", "分类2"]
coverImage: "/blog/你的封面图片.jpg"
---
```

然后把封面图片放到 `public/blog/` 目录即可。

## Markdown 基本语法

### 标题

使用 `#` 表示标题，`##` 是二级标题，以此类推。

### 列表

无序列表用 `-`：
- 第一项
- 第二项

有序列表用数字：
1. 第一步
2. 第二步

### 代码

行内代码用反引号：`console.log('Hello')`

代码块用三个反引号：

```javascript
function hello() {
  console.log('Hello, World!')
}
```
